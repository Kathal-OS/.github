# 🚀 Our Mission

BakeWeb builds open-source infrastructure software that makes deploying, hosting, automating, and managing applications simple, secure, and scalable.

Our flagship product is **KATHAL OS**, an open-source Hosting & Development Operating System inspired by the modular nature of the jackfruit.
